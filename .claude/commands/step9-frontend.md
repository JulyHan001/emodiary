# Step 9: Streamlit 前端 + 联调

## 本步目标
用 Streamlit 构建前端界面，包含聊天窗口、情绪仪表盘和历史日记浏览。

## 前端页面结构
```
┌─────────────────────────────────────────────────┐
│                  前端 (Streamlit)                 │
│  聊天界面 │ 情绪仪表盘 │ 历史日记 │ 成长报告      │
└─────────────────────────────────────────────────┘
```

### 页面详情
1. **聊天界面** — 主页面，对话式记录日记
   - 消息输入框 + 发送按钮
   - 聊天气泡展示对话历史
   - 每条回复显示识别到的情绪标签
2. **情绪仪表盘** — 可视化情绪趋势
   - 最近 7 天情绪折线图
   - 情绪分布饼图
   - 情绪强度热力图
3. **历史日记** — 浏览和搜索
   - 日记列表（按日期排序）
   - 语义搜索框
   - 按情绪/日期筛选
4. **成长报告** — 周报/月报
   - 选择时间范围
   - 生成并展示 markdown 报告

## 实现任务
1. 实现 frontend/streamlit_app.py — 多页面布局
2. 聊天界面：接入 POST /api/chat，展示对话
3. 情绪仪表盘：接入 GET /api/diary，用 plotly/altair 画图
4. 历史日记页面：接入 GET /api/diary + GET /api/search
5. 成长报告页面：接入 POST /api/report，渲染 markdown

## 完成标准
- [ ] `streamlit run frontend/streamlit_app.py` 能启动
- [ ] 聊天界面能正常记录日记并显示 AI 回复
- [ ] 情绪仪表盘图表正常渲染
- [ ] 历史日记页面能浏览和搜索
- [ ] 成长报告能生成并展示
- [ ] 前端与 FastAPI 后端联调通畅
